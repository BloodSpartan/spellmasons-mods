# Spellmasons Mods

## Supported Modding
- Custom Units
- Custom Spells
- Custom Pickups
- Custom Art
- Custom Sound effects

If you want to create a mod that isn't currently supported, please let me know in the #modding channel of the [Spellmasons Discord](https://discord.com/invite/q6sUCreHeJ)
